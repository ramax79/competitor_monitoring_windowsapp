# Desktop App — Мониторинг конкурентов

Десктопное приложение на PyQt6, повторяющее функционал веб-версии.

## Возможности

- ✅ Анализ конкурентного текста
- ✅ Анализ изображений (баннеры, сайты, упаковка)
- ✅ Парсинг сайтов конкурентов через Selenium
- ✅ История запросов с возможностью просмотра
- ✅ Раскрывающиеся секции результатов
- ✅ Современный тёмный UI

## Требования

- Python 3.10+
- Backend должен быть запущен на `http://localhost:8000`

## Установка

1. **Установите зависимости:**

```bash
cd desktop_app
pip install -r requirements.txt
```

2. **Убедитесь, что backend запущен:**

```bash
# В корне проекта
uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000
```

3. **Запустите приложение:**

```bash
python main.py
```

## Сборка в .exe

### Вариант 1: Автоматическая сборка

```bash
python build_exe.py
```

### Вариант 2: Ручная сборка через PyInstaller

```bash
pyinstaller --name=CompetitorMonitor --onefile --windowed main.py
```

Готовый `.exe` файл будет в папке `dist/`.

### Опции сборки

- `--onefile` — один исполняемый файл
- `--windowed` — без консоли (GUI только)
- `--icon=icon.ico` — добавить иконку (опционально)

## Настройка API

По умолчанию приложение подключается к `http://localhost:8000`.

Чтобы изменить адрес, отредактируйте переменную `API_BASE` в `main.py`:

```python
API_BASE = "http://your-backend-url:8000"
```
